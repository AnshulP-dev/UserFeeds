# UserFeeds
<b>UserFeeds</b> app shows the list of User Feeds at Home Screen and on tap of any feed, reveals it in detail screen. It has features to sort User Feeds based on thier content type e.g. All Feeds, Text, Image, Other. Plus users can view their feeds in offline mode as well.

### Installation:
For the sake of easiness, this repo also has Pods pushed in the repo. So users can just download code and run to see the app behavior. i.e. no need to intall CocoaPods, the repo already has required pods.

### Requirement:
* Built with Xcode 12.0
* iOS 13.0 + 

### Following are the 3rd party frameworks used:
1. Alamofire
2. AlamofireImage
3. RealmSwift
4. SnapKit
